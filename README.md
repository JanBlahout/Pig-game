# Pig-game
Game for 2 players.

# How to play
Player clicks on a button "Roll dice". If you roll 2-6, score is added to your current score. To save the score you need to press "Hold".
If you roll 1, you will lose all your current score and second player is playing.
Player who gets 100 points wins.
